# Proof boundary

The certified threshold is restricted to the normalized convex Brass-Sharifi three-test-set certificate model.

Allowed statement:

- The bundled finite certificate verifies the threshold `0.833` in the normalized convex Brass-Sharifi three-test-set framework.
- Within this framework, the certificate gives the convex consequence `alpha_cvx >= 0.833`.

Not claimed:

- A theorem for the unrestricted nonconvex Lebesgue universal cover problem.
- A proof-assistant formalization.
- Completed independent external verification.
- Any stronger threshold beyond the certified threshold in this release.
