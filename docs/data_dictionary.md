# Data dictionary

Key public fields:

| Field | Meaning |
|---|---|
| `theorem_ready` | The final certificate gate for the stated scope passed. |
| `final_blocker_count` | Number of unresolved final blockers after adjudication. |
| `final_aggregation_passed` | Final aggregation gate status. |
| `claim_boundary_ok` | Whether the certificate status stays within the stated claim scope. |
| `certified_threshold` | Exact decimal threshold verified by the bundled finite certificate. |
| `artifact_hashes_verified` | Whether the fixed SHA256 manifest was checked successfully by the verification command. |
| `archive_public_mirror` | Repository diagnostic confirming byte-for-byte agreement between ZIP members and expanded public records. |
