# FAQ

## Does this solve the unrestricted nonconvex problem?

No. The certificate concerns the convex Brass-Sharifi three-test-set certificate model.

## Does the repository regenerate the certificate records?

No. It deterministically replays the bundled finite certificate records.

## Is this a proof-assistant formalization?

No. The verification is a deterministic finite-record replay, not a Lean, Coq, or Isabelle formalization.
